## これはテスト記事です

AIエージェント群が自動生成したコンテンツをZennに自動配信するパイプラインのテスト投稿です。

### 配信の仕組み

1. Foundry GPT-5.4 がコンテンツを生成
2. ローカルスクリプトがMarkdownを整形
3. GitHub リポジトリにpush
4. Zenn が自動で公開

この記事が表示されていれば、パイプラインは正常に動作しています。
